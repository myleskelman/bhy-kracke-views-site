Privacy Policy

Last updated: June 24, 2026

Overview

Bhy Kracke Views is a personal creator tool used by a single operator to upload timelapse videos to social media platforms. It is not a public-facing application and is not available to other users.

Data Collection

This application does not collect, store, or process personal data from any third party. OAuth access tokens obtained during authentication are stored locally on the operator's own device and are never transmitted to any external server other than the platform being authenticated against (e.g., TikTok, YouTube).

Data Sharing

No user data is shared with any third party. The application has no backend server, no database, and no analytics.
